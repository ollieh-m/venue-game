This is a game to play on the command line based loosely on the ‘raise a dragon’ exercise in Chris Pine’s ‘Learn to Program’ (https://pine.fm/LearnToProgram/). 

I wrote it in February 2016 as part of my efforts to learn ruby…

Just run the program from the command line with ‘ruby venue_game.rb’ and follow the instructions.
